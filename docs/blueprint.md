# **App Name**: StackUp Frenzy

## Core Features:

- Tap-to-Drop Mechanic: Players tap the screen to drop blocks onto a rotating tower, aiming for perfect alignment.
- Dynamic Rotating Platform: The base tower continuously rotates, requiring precise timing from the player.
- Score and Combo System: Track the current stack height and award bonus points for perfectly aligned blocks (combos).
- High Score Tracking: Persistently save and display the player's highest score achieved.
- Game Over Logic: Detect when a block misses or falls off the tower, ending the current game session.
- Quick Restart Option: Allow players to quickly restart a new game immediately after a game over.
- AI Dynamic Challenge Adjuster: An AI tool that dynamically adjusts minor gameplay parameters like tower spin speed or block drop interval based on player performance to keep the experience fresh and engaging without explicitly increasing difficulty linearly.

## Style Guidelines:

- Primary color: Vibrant Orange (#FA7C33) to evoke energy and excitement in a playful, hypercasual context.
- Background color: Soft Peach (#FBEAE3), a heavily desaturated variant of the primary hue, ensuring focus remains on the game elements.
- Accent color: Playful Magenta-Pink (#FA72BB), chosen for an analogous contrast that complements the primary color and adds a fun pop for feedback and UI elements.
- Headline and body font: 'Poppins' (sans-serif) for its precise, contemporary, and legible style, ideal for quick readability in a fast-paced game.
- Use simple, bold, and geometric icons that are immediately recognizable and align with the modern hypercasual aesthetic.
- A clean, minimalist, and centered layout, ensuring the main game area and call-to-action buttons are prominent and easy to interact with on mobile devices.
- Incorporate satisfying, snappy animations for block drops, perfect stack feedback, score updates, and transitions to enhance the tactile feel and engagement.
